# Калькулятор геометрии клинка 🔪

Профессиональный инструмент для расчёта углов спусков ножей.

## Возможности
- Расчёт основного угла спуска (α)
- Расчёт угла подводов (β)
- Визуализация профиля в реальном времени
- Автоматический пересчёт при изменении параметров
