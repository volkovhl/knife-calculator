# Калькулятор геометрии клинка 🔪

Профессиональный инструмент для расчёта углов спусков ножей.

## Возможности
- Расчёт основного угла спуска (α)
- Расчёт угла подводов (β)
- Визуализация профиля в реальном времени
- Автоматический пересчёт при изменении параметров

## Как использовать
1.  [Смотреть](https://volkovhl.github.io/knife-calculator/) 
[youtube] (https://m.youtube.com/watch?v=hNq_J7gQe2w) 
2. Введите параметры клинка
3. Получите готовый расчёт и визуализацию
4. Скачайте файл [index.html](https://github.com/volkovhl/knife-calculator/archive/refs/heads/main.zip)  
## Технологии
- HTML5
- CSS3
- JavaScript (Canvas)
