## 📲 Как использовать

1. Откройте **[калькулятор](https://volkovhl.github.io/knife-calculator/)**  
2. Введите параметры клинка (ширина, толщина, спуск и т.д.)  
3. Выберите тип ножа (Кухонный / Охотничий / EDC) для пресетов  
4. Смотрите мгновенную визуализацию и результаты
